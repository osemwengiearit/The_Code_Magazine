# The_Code_Magazine

A structured HTML &amp; CSS fundamentals project focusing on semantic markup, proper document structure, lists, images, links, and clean styling practices.

## 🚀 Project Overview

The goal of this project is to build a simple, well-structured webpage using:

- Semantic HTML elements
- Headings and paragraphs
- Lists (ordered & unordered)
- Images and links
- Basic CSS styling

## 📚 What I Learned

- How to structure an HTML document properly
- How to use semantic tags for clean markup
- How to add images, lists, links, and text formatting
- How to apply simple CSS for layout and styling

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
